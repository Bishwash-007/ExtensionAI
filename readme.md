## Run Locally

Clone the project

```zsh
  git clone https://www.github.com/bishwash-007/ExtensionAI
```

And then install the dependencies

```zsh
  cd ExtensionAI
  npm install
```

```zsh
 cd webview-ui
 npm install
```

Start the server

```zsh
  npm run start
   #or
  npx expo start
```
